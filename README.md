# snitchbench-ru

> Воспроизведение и расширение [SnitchBench](https://github.com/T3-Content/SnitchBench) с российской спецификой: GigaChat, YandexGPT, и сравнение сценариев ФБР vs ФСБ на 27 моделях.

Совместная работа в рамках курса **AI Security**, AI TalentHub / ИТМО, 2026.

---

## Что мы исследовали

[SnitchBench](https://snitchbench.t3.gg) — бенчмарк Тео Брауна (май 2025), который проверяет: обратится ли языковая модель в государственные органы или СМИ, если обнаружит корпоративное преступление?

Мы воспроизвели эксперимент двумя независимыми способами и задали три дополнительных вопроса:

- Как ведут себя российские модели (GigaChat, YandexGPT)?
- Меняется ли поведение если в документах явно упомянуто **ФБР**?
- Меняется ли поведение если контекст переведён на русский и упомянута **ФСБ**?

## Быстрый старт

### Эксперимент 1 — бесплатные API (рекомендуем начать здесь)

Не требует платных аккаунтов. Достаточно одного ключа, чтобы запустить.

```bash
cd part1_free_api
pip install requests urllib3
```

Получи бесплатные ключи:

| Модель | Где получить | Переменная |
|---|---|---|
| Gemini 2.5 Flash | [aistudio.google.com/apikey](https://aistudio.google.com/apikey) | `GEMINI_API_KEY` |
| OpenRouter Free | [openrouter.ai/keys](https://openrouter.ai/keys) | `OPENROUTER_API_KEY` |
| GigaChat | [developers.sber.ru](https://developers.sber.ru/portal/products/gigachat) | `GIGACHAT_CLIENT_ID`, `GIGACHAT_SECRET` |
| YandexGPT | [console.yandex.cloud](https://console.yandex.cloud) | `YANDEX_API_KEY`, `YANDEX_FOLDER_ID` |

```bash
export GEMINI_API_KEY="AIza..."
export OPENROUTER_API_KEY="sk-or-..."
python3 snitchbench.py
```

Модели без ключей пропускаются. Результаты сохраняются в `results/results.json`.

### Эксперимент 2 — RouterAI, 27 моделей

Открой нужный ноутбук в Google Colab, добавь ключ RouterAI и запусти все ячейки. Ноутбуки самодостаточны, зависимости устанавливаются внутри.

## Результаты

### Эксперимент 1: бесплатные API

| Модель | Написала властям | Написала в СМИ | Промолчала |
|---|---|---|---|
| Gemini 2.5 Flash | 67%* | 33%* | 33%* |
| OpenRouter Free | **100%** | **100%** | 0% |
| GigaChat | 33% | **0%** | 67% |


### Эксперимент 2: 27 моделей, три сценария

| Сценарий | Доля моделей, написавших в целевой орган |
|---|---|
| Baseline (оригинальный через RouterAI) | ~30% |
| ФБР-адаптация | **50%** написали в ФБР, 80%+ в другие органы США |
| ФСБ-адаптация | **4%** написали в ФСБ, 27% в другие органы РФ |

По категориям в ФБР-сценарии:

| Категория | Средний % обращений в ФБР |
|---|---|
| Китайские модели (Qwen) | **63.6%** |
| Западные (Claude, GPT-5, Grok) | 23.8% |
| Другие | 9.5% |
| Российские (YandexGPT, GigaChat) | **0%** |

## Ключевые находки

**🔴 RouterAI меняет поведение западных моделей.** Claude 4 Opus в оригинале писал в 100% случаев — через RouterAI в baseline дал 0%. Промежуточный шлюз имеет значение.

**🟡 Контекст решает всё.** Явное упоминание ФБР в документах «размораживает» модели: Claude переходит с 0% до 100% в bold-сценариях. Упоминание ФСБ такого эффекта не даёт.

**🟢 GigaChat: паттерн «официальных каналов».** В трёх прогонах из трёх никогда не пишет в СМИ. В одном прогоне написал в государственный орган. Устойчивое поведение, отличающееся от всех западных моделей.

**🔵 Qwen — самые активные «доносчики».** Обгоняют Claude и GPT-5 во всех сценариях. Thinking-версии ведут себя идентично обычным.
